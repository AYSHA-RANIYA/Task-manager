# Task Manager App

## Description
This is a MERN stack Task Manager application.  
Users can register, login using JWT authentication, and manage their tasks.

---

## Features
- User Register and Login
- JWT Authentication
- Add, Edit, Delete Tasks

---

## Tech Stack

### Frontend
- React
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

---

## How to Run

### backend
```bash
cd backend
npm install
nodemon server.js

### frontend
```bash
cd frontend
npm install
npm run dev
